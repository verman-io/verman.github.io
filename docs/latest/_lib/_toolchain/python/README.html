_lib/_toolchain/python

Python installer.

Usage

  • UNIX
  • Windows
$ # Replace `$(pwd)` if not in the 'libscript' directory or a generated one.
$ export LIBSCRIPT_ROOT_DIR="$(pwd)"
$ # `export` any variables now (see below for variable docs). Then:
$ # Set script location:
$ export SCRIPT_NAME="${LIBSCRIPT_ROOT_DIR}"'/_lib/_toolchain/python/setup.sh'
$ . "${SCRIPT_NAME}"
> :: Replace `%cd%` if not in the 'libscript' directory or a generated one.
> SET "LIBSCRIPT_ROOT_DIR=%cd%"
> :: `SET` any variables now (see below for variable docs). Then:
> :: Set script location:
> SET SCRIPT_NAME="%LIBSCRIPT_ROOT_DIR%\_lib\_toolchain\python\setup.cmd"
> CALL "%SCRIPT_NAME%"

## python toolchain vars

python toolchain vars that can be set

python toolchain vars Properties

Type Description Required
PYTHON_VERSION string Version of Python demanded. Needs to be a specific numeric version number No
VENV string Path to a Python virtualenv (will create if nonexistent) No

Additional properties are allowed.

python toolchain vars.PYTHON_VERSION

Version of Python demanded. Needs to be a specific numeric version number

  • Type: string
  • Required: No

python toolchain vars.VENV

Path to a Python virtualenv (will create if nonexistent)

  • Type: string
  • Required: No
