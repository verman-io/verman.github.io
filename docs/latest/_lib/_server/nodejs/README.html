$ # Replace `$(pwd)` if not in the 'libscript' directory or a generated one.
$ export LIBSCRIPT_ROOT_DIR="$(pwd)"
$ # `export` any variables now (see below for variable docs). Then:
$ # Set script location:
$ export SCRIPT_NAME="${LIBSCRIPT_ROOT_DIR}"'/_lib/_server/nodejs/setup.sh'
$ . "${SCRIPT_NAME}"

## nodejs server vars

nodejs server vars that can be set

nodejs server vars Properties

Type Description Required
DEST string Destination (working directory) No
VARS string Key/value in JSON format (as an escaped string) No

Additional properties are allowed.

nodejs server vars.DEST

Destination (working directory)

  • Type: string
  • Required: No

nodejs server vars.VARS

Key/value in JSON format (as an escaped string)

  • Type: string
  • Required: No
